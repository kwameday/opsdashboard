# Telecel Marketplace Operations Dashboard

An advanced operational tracker for merchant acquisition and onboarding.

## 🚀 Deployment
1. Upload `index.html` to a new GitHub repo.
2. Link the repo to Cloudflare Pages.
3. Your dashboard is now live!

## 📊 Updating Data
- **KPIs:** Edit lines 48-60 in `index.html`.
- **Categories:** Edit the icon tags and counts in lines 79-144.
- **Trend Chart:** Edit the data array in line 181.

## 🛠 Features
- **Lucide Icons:** Visual category identification.
- **Chart.js:** Dynamic MoM growth tracking.
- **Tailwind CSS:** Fully responsive, modern mobile-first design.
